How to use this tool:

  1. Click "Choose Excel File" to upload your transaction history Excel file
  2. Ensure your Excel file has columns for Transaction ID, Location, Plate, and Transaction Amount
  3. Use the search box to find transactions for a specific license plate
  4. Click on any plate number to quickly search for that plate
  5. The application will show the total transaction amount for the searched license plate
